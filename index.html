Reflexão e Aplicação Prática

Propriedades CSS usadas:

  • display: flex – Ativa o Flexbox no container.
  • justify-content – Alinha os itens na horizontal.
  • align-items – Alinha os itens na vertical.
  • flex-direction – Define a direção dos itens (linha ou coluna).
  • align-self – Alinha um item individual no container.

Aplicação prática:

Eu usaria Flexbox para criar um menu de navegação horizontal que se adapta a diferentes tamanhos de tela. Também poderia aplicar em uma galeria de produtos, alinhando os cards responsivamente com espaçamento igual.
